rna\_convert\_pseudoknot\_formats
--------------------------------------------------------------------------
Run this as:

	python rna-pk-simrna-to-one-line.py test_data/simrna.ss

Convert 

	> a
	....((.(..(((....)))..((((.(.........).)))....).)).......((((......))))
	..............................((((...................))))..............

to 

	> a
	....((.(..(((....)))..((((.(..[[[[...).)))....).))...]]]]((((......))))

and

	> b
	..(.......(((....)))..((((.(.........).))))).............((((......))))
	....((.(......................................).)).....................
	..............................((((...................))))..............

to

	> b
	..(.[[.[..(((....)))..((((.(..<<<<...).)))))..].]]...>>>>((((......))))
